## 🏢 About AptBites

AptBites is a Next.js-based convenience store application designed specifically for apartment communities. Built for a 400-apartment community in Fort Bend County, Texas.

**Key Features:**
- 🛒 Full shopping cart functionality
- 📱 Multiple order methods (SMS, WhatsApp, Email)
- 🏠 Free delivery within community
- 💰 Cash, Venmo, and Zelle payment options
- 🕐 8:00 AM - 8:00 PM delivery hours
- 🔐 Password-protected admin dashboard
- 📦 Real-time product management

---

## 🛠️ Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **State Management:** React Context API
- **Storage:** localStorage
- **Deployment:** Vercel (recommended)
